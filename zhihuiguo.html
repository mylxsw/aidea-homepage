什么是智慧果

“智慧果”是指用户在 AIdea 账户中的虚拟资产。
我们期待该应用能给予我们启发和帮助，因此命名为“智慧果”。

智慧果使用规范

  1. 智慧果仅可在 AIdea 应用中供账号所有者使用，不支持退款、提现或转赠他人；
  2. 我们会根据您使用的服务及模型复杂度分别计费，详情参见下文“计费规则”。

如何获得智慧果

您可通过以下方式获得智慧果：

  1. 注册赠送： 首次成功注册 AIdea 账号后，即可获得 30 个智慧果奖励；
  2. 单次购买： 您可前往 “AIdea-设置-智慧果” 页面进行购买；购买成功后将下发至您的“智慧果”账户；
  3. 邀请好友：
  1. 每次邀请新用户注册成功，您和好友都将获得 100 个智慧果作为奖励，成功邀请多人奖励可累积；
  2. 好友注册成功，1 年内每购买智慧果，您将获得好友购买数量 *5% 的智慧果作为奖励；
  1. 其他福利活动

计费规则

1. 文本类计费逻辑：

在“聊一聊”、“顾问团”、“创作岛”中使用文本对话服务，将依据模型复杂度及“输入 Token 数+输出 Token 数”总和计费。

注：文本对话服务因涉及上下文关联，Token 总数也包含上文输入和输出的 Token 数。

1）什么是 Token：

Token 通常由单词、标点符号或其他语言单位组成。它是大语言模型中处理文本的基本单元。我们输入的文本，会以 Token（词元）的形式存在于模型内部，被组合、转换或调整，以执行某个任务。

以 Token 作为计费单位，不仅可以降低用户的实际使用成本，还能够更加精准的反应出用户真实消费情况，从而最大程度地保证计费模式的公平合理性。

【举例】

  • “人工智能” 这四个字的 Token 数为 8，其中“人”的 Token 数为 1；“工”的 Token 数为 2；“智”的 Token 数为3；“能”的 Token 数为 2；
  • “AI” 这个单词的 Token 数为 1；
  • “GPT” 这个单词的 Token数为 2；

根据粗略的经验，对于中文文本，平均 1 个中文字约为 2 个 Token；对于英文文本，1 个 Token 约为 0.75 个单词。当然，您也可以使用 Token 计数器 https://platform.openai.com/tokenizer ，来看看他们是如何被计算的。

有效期

  1. 智慧果具有有效期，有效期以具体购买规则及活动规则为准；
  2. 不同有效期的智慧果可以合并使用，使用时将优先扣除您即将过期的智慧果；
  3. 您可通过“AIdea-设置-购买-购买历史”查看具体有效期，逾期未使用即失效，请在有效期内尽快使用。
